# Mega-Project-1---Jarvis
A smart, voice-activated personal assistant built with Python that can listen, understand commands, speak responses, fetch news, open websites, and use Google Gemini (GenAI) for AI-powered conversations. Includes wake-word detection (“Jarvis”), speech recognition, text-to-speech, API integrations, and real-time audio output using gTTS + Pygame.
